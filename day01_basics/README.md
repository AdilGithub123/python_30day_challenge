# Day 1 — Python Basics Projects

This folder contains three beginner-level Python projects completed on Day 1 of my 30-day Python backend training journey.

## Projects
1. **Grade Calculator**
   - Inputs 3 exam scores, validates them, and outputs the average and letter grade.

2. **Simple Interest Calculator**
   - Computes simple interest and total amount based on principal, rate (%), and time.

3. **Unit Converter (KM ↔ Miles)**
   - Converts values between kilometers and miles with input validation.
